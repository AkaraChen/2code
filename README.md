## Why 2code?

Every developer knows the feeling: you're deep in flow, making progress, and then—*alt-tab*. Terminal here. IDE there. AI chat somewhere else. Git history in another window. Each switch is a tiny context tax that adds up to lost momentum.

**2code is built on a simple idea: your coding environment shouldn't break your flow.**

It's a desktop workspace where terminal, AI agents, and git live together—designed specifically for the emerging practice of **vibe coding**: the art of staying in flow while collaborating with AI.

---

## The Philosophy

### Two Coders, One Workspace

The name "2code" reflects its core vision: **you + AI, working as peers**.

In 2code, AI agents aren't crammed into a sidebar or modal. They have their own tabs, right alongside your terminals. Your AI partner has persistent memory, full conversation history, and stays connected to your project context.

### Never Lose Context

- **Persistent terminals** that survive app restarts with full scrollback history
- **Agent conversations** that remember where you left off
- **Git state** that's always visible, always current

The app remembers everything so you don't have to.

### Stay in Your Lane

Working on multiple features? Each **profile** creates an isolated git worktree with its own terminal sessions and agent conversations. No more switching branches and losing your terminal history or AI context.

### Flow State, Engineered

Every design decision in 2code serves one goal: **minimize context switching**.

- Real-time diff viewer always accessible
- Activity heatmap that gamifies consistent coding
- Audio notifications for long-running commands
- Keyboard-first navigation throughout

## What You Can Do

**Projects & Profiles**
- Import any code folder as a project
- Create isolated worktrees for each feature or experiment
- Automatic setup/teardown scripts per profile

**Terminals**
- Multiple tabs per profile
- Customizable themes, fonts, and sizes
- Sessions persist across restarts
- Audio alerts when commands complete

**AI Agents**
- Built-in support for Claude Code, Codex, Cursor, and more
- Agent marketplace to discover new AI tools
- Rich chat UI with tool call visualization
- Switch models within the same conversation

**Git Integration**
- Live diff viewer for staged and unstaged changes
- Commit history browser with syntax highlighting
- One-click pull request creation
- GitHub Desktop integration

## Who It's For

2code is for developers who:

- Work with AI coding assistants daily
- Live in the terminal but need more than just a shell
- Juggle multiple feature branches simultaneously
- Believe the best development environment is one you don't notice

## The Vision

We believe the future of software development is **human + AI collaboration**—not AI replacing developers, but AI as a persistent partner that understands your codebase, remembers your conversations, and works alongside you in the same space.

2code is our attempt to build that space.
